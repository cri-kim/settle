# 정산시스템 파일럿 프로젝트
## 개요
우아한 형제들의 기술 블로그에 올라온 신입 직원들의 pilot 프로젝트를 참고하여 프로젝트 진행

## 도메일 설계 및 개발
### 기능 요구사항
- 어드민(Member)
  - 일반회원/관리자
  - 회원가입/로그인/권한관리 기능
- B2B 회원(Owner)
- 주문 관리 기능(Order)
- 보상 금액 기능(Reward)
- 지급 관리 기능(Settle)

### 기술 요구사항
- OOP & Clean Code
- 단위 테스트 & 통합테스트
- OWSAP TOP 10
- HTTP API
- JPA
- Gradle
- Lombok
